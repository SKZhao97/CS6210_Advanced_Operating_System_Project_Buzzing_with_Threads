1. Type 'make' to compile the GTThreads library 
2. Type 'make matrix' to compile the matrix program
3. ./bin/matrix -c 0 to run the matrix program in O(1) scheduler
4. ./bin/matrix -c 1 to run the matrix program in credit scheduler
5. ./bin/matrix -c 1 -l 1 to run the matrix program in credit scheduler with load balancing
6. ./bin/matrix -c 1 -y 1 to run the matrix program in credit scheduler with yield
